# Lernfeld 08 – Daten systemübergreifend bereitstellen

In diesem Lernfeld geht es darum, Daten über Systemgrenzen hinweg nutzbar zu machen und Softwarelösungen methodisch zu planen und umzusetzen.

## Lernziele (Kurzfassung)

- Projekt vs. Routineaufgaben sicher unterscheiden
- Klassische und agile Vorgehensmodelle vergleichen (Scrum, Kanban, XP)
- Kriterien guter Datenqualität kennen und anwenden
- Formen von Datenintegration (physisch, virtuell, Data Warehouse, Data Lake) erklären
- Wichtige Programmierparadigmen (imperativ, prozedural, funktional, logisch) zuordnen
- UML-Diagramme (Use Case, Klassen, Aktivität) lesen und einfache Beispiele erstellen
- Grundlagen von Java (Datentypen, Variablen, Verzweigungen, Schleifen, Arrays) beherrschen
- OOP-Konzepte in Java anwenden (Klassen, Objekte, Methoden, Konstruktoren, Vererbung, Kapselung, static, abstract)
